# Naming Direction

This project started as **Evercrew Local Telegram Task Bot**.

During planning, we identified a broader product direction:

# EVE

**EVE** is the product platform concept created by **Evercrew Venture Pte Ltd**.

EVE means:

- **Evercrew Virtual Engine**
- The beginning before something new starts, like New Year's Eve
- The starting point of practical AI adoption for SMEs

## Brand Meaning

Every great transformation begins with an eve.

Before a new year begins, there is New Year's Eve.

Before a business changes, there is a decision to begin.

EVE represents that beginning.

It is designed to help SMEs take their first practical step into AI workflow automation.

## Product Direction

EVE is intended to become a local-first AI workflow platform for SMEs.

Possible future components:

- EVE Core
- EVE Telegram
- EVE WhatsApp
- EVE Email
- EVE Task Assistant
- EVE Meeting Assistant
- EVE Invoice Assistant
- EVE Admin Assistant
- EVE Sales Assistant
- EVE Customer Assistant

## Current Repository Role

This repository remains the first public proof-of-concept:

Boss instruction → Local LLM → Confirmation → Employee task

Future repositories may separate EVE Core from individual connectors and workflows.
