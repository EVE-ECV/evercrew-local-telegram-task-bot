============================================================
STEP 0 - Before Installing EVE
==============================

You should have already:

✓ Downloaded EVE from:

https://github.com/EVE-ECV/EVE

✓ Clicked:

Code
↓

Download ZIP

✓ Extracted the ZIP file.

You should now have a normal folder called something similar to:

EVE-main

If you are still viewing the GitHub website,
please download and extract EVE first.

============================================================
STEP 1 - Install Python
=======================

Download Python from:

https://www.python.org/downloads/

During installation, make sure you tick:

[✓] Add Python to PATH

Then complete the installation.

If Python is already installed,
you can skip this step.

============================================================
STEP 2 - Install Ollama
=======================

Download Ollama from:

https://ollama.com/download

Install using the default settings.

If Ollama is already installed,
you can skip this step.

============================================================
STEP 3 - Download the AI Model
==============================

Open Command Prompt.

Type:

ollama pull llama3.2:3b

Wait for the download to complete.

This only needs to be done once.

============================================================
STEP 4 - Install EVE
====================

Inside the EVE folder, double-click:

Install EVE.bat

------------------------------------------------------------
IMPORTANT FOR WINDOWS 11 USERS
------------------------------------------------------------

Windows may display a message such as:

"Smart App Control blocked a file that may be unsafe."

This does NOT necessarily mean EVE is unsafe.

Windows blocks many downloaded batch (.bat) files because they
came from the Internet.

If you downloaded EVE directly from:

https://github.com/EVE-ECV/EVE

then this warning is expected during the Alpha release.

If Windows blocks the installer:

1. Right-click Install EVE.bat
2. Click Properties
3. If you see an "Unblock" checkbox, tick it.
4. Click Apply.
5. Click OK.
6. Double-click Install EVE.bat again.

If you do not see an "Unblock" option,
please refer to QUICK START.pdf or visit:

https://evercrew.ai

------------------------------------------------------------

The installer will automatically:

• Check Python
• Check Ollama
• Install the required Python packages
• Create the .env file (if needed)

The installer will guide you through the setup automatically.

============================================================
STEP 5 - Create Your Telegram Bot
=================================

Open Telegram.

Search for:

BotFather

Create a new bot by following BotFather's instructions.

Copy your Bot Token.

Keep it safe.

============================================================
STEP 6 - Configure EVE
======================

Open:

.env

Replace the placeholder values with your own settings.

At minimum, update:

BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN

Save the file.

============================================================
STEP 7 - Add Your Employees
===========================

Open:

data\employees.json

Replace the sample employees with your own employee names and Telegram Chat IDs.

Save the file.

============================================================
STEP 8 - Start EVE
==================

Double-click:

Start EVE.bat

If everything is configured correctly,
EVE will start and wait for Telegram messages.

IMPORTANT

Do NOT close the Command Prompt window while EVE is running.

Closing the window will stop EVE.

============================================================
STEP 9 - Test EVE
=================

Open Telegram.

Send a message to your EVE bot, for example:

Ask Ah Tan to prepare the June sales report by Friday.

EVE will:

• Understand your request
• Confirm the task with you
• Assign the task to your employee
• Notify you when the task is completed

Congratulations!

Your first local AI workflow is now running.
